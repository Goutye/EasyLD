# EasyLD

Project to allow the programmer in Löve2D or Drystal (Other game engine in the future) to programm a game as easily as possible.

Current features :
-----------------

* Tileset
* Map
* MapEditor
* Vector / Box / Point / Collide

In the side of LÖVE2D :
----------------------

* ImageAdapter
* Keyboard
* Mouse
