# EasyLD

Project to allow the programmer (who uses Löve2D or Drystal) to programm a game as easily as possible .

Current features :
-----------------

* Tileset
* Map
* MapEditor
* Matrix / Vector / Point
* Collide (OBB, AABB, Circle, Point)
* Box
* Keyboard
* Mouse

In the side of LÖVE2D :
----------------------

* Circle fill/line

----------------------
